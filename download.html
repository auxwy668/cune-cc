常见问题

Windows 7 能装吗？ ▾

Telegram 官方已不再支持 Windows 7，但你可以尝试安装 v4.13.0 或更早版本。不过，为了安全和功能考虑，建议升级到 Windows 10 或以上。

安装后无法打开怎么办？ ▾

可能是系统缺少 Visual C++ 运行库。请下载并安装 Visual C++ Redistributable，然后重新启动 Telegram。

如何彻底卸载 Telegram？ ▾

进入"设置" → "应用" → 找到 Telegram → 点击"卸载"。如果想彻底清除聊天记录，还需要删除 %AppData%\Telegram Desktop 文件夹。
